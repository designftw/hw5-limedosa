    • Heuristic Evaluation:

    Overall this app is a very pleasing user experience. This app fails Nielsen's heuristic principle of Constency and Standards, due to its lack of otimization for web usage. The page doesn't stretch out horizontally, as pages optimized for web usage should and uses a more rectangular framing for the app. The layout is not adaptable to horizontal screens, leading to html issues down the road. One apparent case is in the picture above, where all the details text is cramped up and nearly impossible to read

    Another of Nielsen's heuristics that fails is the Visibility of System Status. The accounts don't get updated. No feedback to the user can imply a non-functional app, even if the spreadsheet does get updated. It is unclear if this is always the case, or if these are isolated instances.

    Flexibility and Efficiency of Use, another one of Neilson's principles also fails. There's an unecessary horizontal scroll and the content becomes off centered when it's interacted with. Horizontal scroll bars are to be avoided whenever possible.

    Lastly, it's important to mention how aesthetically pleasing this app is as well as how easy it is for users to interact and learn how to use the app. Neilson's Aesthetic & Minimalist Design principle is portrayed wonderfully in this app. The design is simple, but beautiful and user friendly. The buttons given constant feedback, adding a lot to the design aspect of this project.
